## CMS 3

__Question__

```
What is the IP address for the Daily Bruin site?

Give the IPv4 version: xxx.xxx.xxx.xxx
(Leading zeroes are omitted)
```

__Hint__

_None_

__Answer__

```
128.97.251.212
```

__Points__

```
5
```

## Solution

This is easily found using the `dig` utility: 

    ...
    ;; ANSWER SECTION:
    dailybruin.com.     3599    IN  A   128.97.251.212
    ...
